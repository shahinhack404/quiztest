Google Spreadsheet Setup (3 Easy Steps)

Step 1: Create Spreadsheet & Share

1. Create a new Google Spreadsheet.

2. In the Spreadsheet Share menu, add the Service Account Client Email as an Editor.

Step 2: Setup Teacher Password

1. Inside the spreadsheet, create two sheets: Quizzes and Results.

2. Set the Username in cell D444 and Password in cell E444 of the Quizzes sheet.

Step 3: Apps Script Method

1. From the spreadsheet menu, go to Extensions -> Apps Script.

2. Clear any existing code and paste the content of the /web/Code.gs file.

3. Click Deploy -> New deployment, select Web app as the type.

4. Select Anyone for "Who has access" and click Deploy. Paste the resulting Web App URL into the alternative URL input above and click save.
